# Simulador de Orçamento Freelancer de TI

### Descrição
Este projeto consiste em um simulador de orçamento de serviços de TI, onde o usuário pode selecionar diferentes serviços e visualizar o valor total atualizado em tempo real.
A aplicação foi desenvolvida com foco em praticar conceitos fundamentais de desenvolvimento web, simulando um cenário real de contratação de serviços freelancers.

### Ferramentas utilizadas 
- HTML
- CSS3
- JavaScript

### Conceitos utilizados
 - Uso de variáveis (let ou const) para armazenar o valor total como uma "caixa" de dados.
 - Manipulação de eventos (ex: onClick) para escutar as ações do usuário.
 - Uso do parseFloat() ou parseInt() para converter os valores recebidos do HTML antes de realizar a soma.
 - Implementação de Arrow Function () => { } para aplicar a estrutura enxuta no código.
